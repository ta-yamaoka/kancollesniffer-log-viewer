# KancolleSniffer Log Viewer

[KancolleSniffer](https://kancollesniffer.bitbucket.io/) が出力する CSV ログを、ブラウザ上で集計・可視化するツールです。

🔗 **[ツールを開く](https://ta-yamaoka.github.io/kancollesniffer-log-viewer/)**

---

## 機能

### ⚙️ 開発報告書

- 総開発数・成功率・失敗数のサマリー
- 種別ごとのドーナツチャート
- 日別開発数の折れ線グラフ（直近30日）
- 装備別・秘書艦別ランキング
- レシピ別開発数ランキング（成功率付き）
- **レシピ別装備開発確率分布**
  - レシピをドロップダウンで選択 or 数値入力
  - 秘書艦チェックボックスで絞り込み（選んだ期間・レシピの中に存在する艦のみ表示）
  - 横棒グラフ＋全装備確率テーブル（種別カラー付き）

### ⚓ 海戦・ドロップ報告書

- 総出撃数・ドロップ率・S勝利率のサマリー
- マップ別出撃数バーチャート・勝利ランク分布
- ドロップ艦娘ランキング TOP30
- **マップ・マス別ドロップランキング**（ボス/道中を色分け表示）
- 期間・マップ・マス（ボスのみ / 道中のみ）・ランク・艦娘名でフィルター
- 特定艦娘のマップ別ドロップ率検索

---

## 使い方

1. 上のリンクからツールを開く
2. KancolleSniffer が出力した CSV ファイルをドラッグ＆ドロップ
   - 開発報告書タブ → `開発報告書.csv`
   - 海戦・ドロップタブ → `海戦・ドロップ報告書.csv`
3. フィルターを操作して集計結果を確認

> CSVファイルはすべてブラウザ内で処理されます。サーバーへの送信は一切ありません。

## 対応ファイル

| ファイル名 | 対応 |
|---|---|
| 開発報告書.csv | ✅ |
| 海戦・ドロップ報告書.csv | ✅ |
| 建造報告書.csv | 未対応 |
| 改修報告書.csv | 未対応 |
| 遠征報告書.csv | 未対応 |
| 資材ログ.csv | 未対応 |

文字コードは **Shift-JIS (CP932)** に対応しています。

---

## 動作環境

モダンブラウザ（Chrome / Firefox / Edge / Safari）で動作します。インストール不要です。

---

## 注意事項

- 非公式の個人制作ツールです。KancolleSniffer および「艦これ」運営とは無関係です。
- ゲームへの介入は一切行いません。ローカルの CSV ファイルを読み込むだけです。
